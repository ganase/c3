## 実CLI接続

コマンドプロンプトでこのフォルダへ移動し、次を実行します。

```bat
run.bat
```

既定ではローカルサーバーを `http://127.0.0.1:5173/` で起動し、作業ルートを `workspace` にします。

```bat
set CODEUI_ROOT=C:\path\to\apps
set CODEUI_PORT=5173
run.bat
```

CLIコマンドは環境変数で差し替えできます。

```bat
set CODEUI_CODEX_COMMAND=codex
set CODEUI_CODEX_ARGS=exec --skip-git-repo-check --sandbox workspace-write
set CODEUI_CLI_TIMEOUT_MS=600000
set CODEUI_CLAUDE_COMMAND=claude
set CODEUI_CLAUDE_ARGS=-p
run.bat
```
